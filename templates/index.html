Configuration HuggingFace

Pour utiliser la diarization des locuteurs, vous devez configurer un token HuggingFace.

Étapes de configuration :

  1. Créez un compte sur HuggingFace (gratuit)
  2. Acceptez les conditions des modèles :
    • speaker-diarization-3.1
    • segmentation-3.0
    • speaker-diarization-community-1 (souvent nécessaire)
  3. Créez un token sur huggingface.co/settings/tokens
    Token "Read" ou Permissions : Read access to contents of public gated repos
  4. Collez le token ci-dessous (commence par hf_)
